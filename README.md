# Algoritmos-de-Substituicao-de-Paginas
Algoritmo em Python para o gerenciamento da memória virtual para Sistemas Operacionais. 
Algoritmos: FIFO - First In, First Out; OTM - Ótimo e LRU - Least Recently Used

Modelo do arquivo de entrada .txt:

A entrada é composta por uma série números inteiros, um por linha, indicando, primeiro aquantidade de quadros disponíveis na memória
RAM e, em seguida, a sequência de referências à memória.

ex.:
número de quadros: 4
sequência da memória: 1 2 3 4 1 2 5 1 2 3 4 5

entrada:
4
1
2
3
4
1
2
5
1
2
3
4
5

Descrição da saída:

A saída é composta por linhas contendo a sigla de cada um dos três algoritmos e a quantidade de faltas de página obtidas com a utilização de cada um deles.
